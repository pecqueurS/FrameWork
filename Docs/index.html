Controller.php

<?php

namespace ads\Controllers;

use Bundles\Formulaires\Forms; /* Inclusion de la classe à utiliser */
class Contact {
	
	public function show() {
		$response = array();

		$forgotPwdForm = Forms::make('Contact'); /* Creation du formulaire d'après le json */
		if (!$forgotPwdForm->isValid()) { /* Vérification du formulaire en fonction des contraintes */
			$response['formContact'] = $forgotPwdForm->render(); /* Création du HTML à afficher */
		} else {
			echo "formulaire valide !"; /* Traitement à effectuer */
		}
	
		return $response;
	}
}

3. Contraintes

Contraintes de bases

NotBlank

true/False

Le champs ne doit pas être vide

Blank

true/False

Le champs doit être vide

NotNull

true/False

Le champs ne doit pas être null

Null

true/False

Le champs doit être null

True

true/False

Le champs doit etre true

False

true/False

Le champs doit etre false

Type

is_*/ctype_*/instanceof *

le champs doit repondre true à la fonction correspondante ex:is_numeric

Contraintes Chaines de caracteres

Email

true/false

Le champs doit être un email

Length

min/max

La longueur du champs doit etre compris entre 2 nombres

Url

true/false

Le champs doit correspondre à un email

Regex

match/pattern

Le champs doit correspondre à la regex utilisée

Ip

version

Le champs doit correspondre à une IP suivant l'une des versions choisie

Contraintes comparatives

Range

min/maxSize

Le nombre doit être compris entre 2 nombres

EqualTo

comparative array / ==

Le champs doit être egal à la contrainte

NotEqualTo

comparative array / !=

Le champs ne doit pas être egal à la contrainte

IdenticalTo

comparative array / ===

Le champs doit être identique à la contrainte

NotIdenticalTo

comparative array / !==

Le champs ne doit pas être identique à la contrainte

LessThan

value to compare / <

Le champs doit être plus petit que la contrainte

LessThanOrEqual

value to compare / <=

Le champs doit être plus petit ou égal que la contrainte

GreaterThan

value to compare / >

Le champs doit être plus grand que la contrainte

GreaterThanOrEqual

value to compare / >=

Le champs doit être plus grand ou égal que la contrainte

Contraintes sur les dates

Date

format

La date doit être une date valide

Contraintes sur les fichiers

File

maxSize/mimeTypes

Vérifie le fichier en fonction du mime et de la taille
